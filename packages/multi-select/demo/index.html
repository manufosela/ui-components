Multi Select

Dropdown with multiple checkbox selections

Basic Usage

Select multiple options from a dropdown

Selected: none
<multi-select id="basic"></multi-select> <script> const select = document.getElementById('basic'); select.options = [ { value: 'js', label: 'JavaScript' }, { value: 'ts', label: 'TypeScript' }, { value: 'py', label: 'Python' } ]; </script>

With Pre-selection

Initialize with some options already selected

Selected: React, Vue
select.options = [...]; select.selectedValues = ['react', 'vue'];

Programmatic Control

Select all or clear selections programmatically

Selected: none
select.selectAll(); // Select all options select.clearAll(); // Clear all selections

Custom Placeholder

Customize the placeholder text

<multi-select placeholder="Choose your skills..."></multi-select>

Disabled State

Disable the entire select

<multi-select disabled></multi-select>

Custom Styling

Use CSS custom properties to style

multi-select { --multi-select-border-color: #8b5cf6; --multi-select-border-focus: #7c3aed; --multi-select-checkbox-color: #8b5cf6; --multi-select-option-selected-bg: #ede9fe; }
