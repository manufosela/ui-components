Slide Notification

Slide-in notifications from the right edge

Notification Types

Four built-in types with appropriate colors and icons

import { showSlideNotification } from '@manufosela/slide-notification'; showSlideNotification({ type: 'success', title: 'Success!', message: 'Operation completed successfully.' });

With Title

Add a title above the message

showSlideNotification({ type: 'info', title: 'New Update', message: 'A new version is available!' });

Custom Duration

Control how long the notification stays visible

showSlideNotification({ message: 'Quick notification', timetohide: 1000 // 1 second }); showSlideNotification({ message: 'Persistent notification', timetohide: 0 // Never auto-hides });

HTML Content

Messages support HTML formatting

showSlideNotification({ type: 'info', message: '<strong>Bold</strong> and <em>italic</em> text' });

Custom Color

Override the default colors

showSlideNotification({ message: 'Custom colored notification', backgroundColor: '#8b5cf6' });

Programmatic Control

Create and control notifications manually

const notification = showSlideNotification({ message: 'Manual notification', timetohide: 0 }); setTimeout(() => notification.hide(), 2000);
