Range Selection

Select a start and end date. Hover over days to preview the range before confirming.

Click a start date, then an end date...
<calendar-inline mode="range"></calendar-inline> <script> cal.addEventListener('range-select', (e) => { console.log('Start:', e.detail.start); // "2025-06-10" console.log('End:', e.detail.end); // "2025-06-20" }); </script>

Month / Year Navigation

Click the month name in the calendar header to open the month picker. Click the year number to open the year/decade picker. Press Escape or select an item to go back to the day grid.

View: days
<!-- Click month or year in the header to switch views --> <calendar-inline></calendar-inline> <script> cal.addEventListener('view-change', (e) => { console.log('View switched to:', e.detail.view); // 'days' | 'months' | 'years' }); cal.addEventListener('month-change', (e) => { console.log('Month:', e.detail.month, 'Year:', e.detail.year); }); // Programmatic navigation cal.goToMonth(2025, 11); // December 2025 (0-based month) cal.goToToday(); </script>

Keyboard Navigation

Click any date in the calendar below, then use the keyboard to navigate.

Key Action
← →Move focus one day back / forward
↑ ↓Move focus one week back / forward
HomeFirst day of the current week
EndLast day of the current week
PageUp / PageDownPrevious / next month
Shift + PageUp/DownPrevious / next year
Enter / SpaceSelect the focused date
EscapeCancel range in progress; return to day view from month/year picker

Form Integration

calendar-inline is a form-associated custom element. It appears in FormData and participates in form validation just like a native input.

FormData will appear here after submit...
<form id="bookingForm"> <label> Pick a date <calendar-inline name="date"></calendar-inline> </label> <button type="submit">Submit form</button> </form> <script> document.getElementById('bookingForm').addEventListener('submit', (e) => { e.preventDefault(); const data = new FormData(e.target); console.log('date:', data.get('date')); // "2025-06-15" }); </script>
