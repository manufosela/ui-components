App Modal

Feature-rich modal with configurable buttons and content

Basic Modal

Simple modal with title and message

import { showModal } from '@manufosela/app-modal'; showModal({ title: 'Welcome', message: 'This is a basic modal dialog.', button1Text: 'OK' });

Confirmation Dialog

Modal with confirm and cancel buttons

showModal({ title: 'Delete Item?', message: 'Are you sure you want to delete this item?<br>This action cannot be undone.', button1Text: 'Delete', button2Text: 'Cancel', button1Action: () => { // Handle delete alert('Item deleted!'); } });

Three Button Modal

Modal with three action buttons

showModal({ title: 'Unsaved Changes', message: 'You have unsaved changes. What would you like to do?', button1Text: 'Save', button2Text: 'Discard', button3Text: 'Cancel', button1Css: 'background: #4caf50;', button2Css: 'background: #f44336;', button3Css: 'background: #6b7280;' });

Custom Content

Modal with custom HTML content element

const form = document.createElement('form'); form.innerHTML = '<input placeholder="Name"><input placeholder="Email">'; const modal = showModal({ title: 'Contact Form', button1Text: 'Submit', button2Text: 'Cancel' }); modal.setContent(form);

No Header/Footer

Minimal modal with only content

showModal({ message: '<h2>Custom Content</h2><p>A minimal modal without header or footer.</p>', showHeader: false, showFooter: false, maxWidth: '300px' });

Custom Size

Modal with custom dimensions

showModal({ title: 'Large Modal', message: 'This modal has custom dimensions.', maxWidth: '700px', maxHeight: '80vh', button1Text: 'Close' });
