Loading Layer

Full-screen loading overlay with spinner, multi-phase messages, and completion events.

← Back to components GitHub Repo Playground npm

Basic Usage

Show a loading overlay with spinner (auto-hides in 3s)

<loading-layer id="loader" timeout="3"></loading-layer> <script> document.getElementById('loader').show(); </script>

With Message

Display a message below the spinner

<loading-layer timeout="3" message="Loading data, please wait..." ></loading-layer>

Custom Colors

Customize spinner color and size

<loading-layer color="#22c55e" size="80" stroke-width="6" message="Almost done..." timeout="3" ></loading-layer>

Different Timeouts

Auto-hide after specified seconds

<loading-layer timeout="2" message="Hiding in 2 seconds..." ></loading-layer>

Global Events

Trigger loading via document-level custom events. Use detail.id to target a specific instance.

Waiting for events...
// Show a specific loading-layer by id document.dispatchEvent( new CustomEvent('loading-layer-show', { detail: { id: 'global', message: 'Triggered via global event!' } }) ); // Hide all / specific document.dispatchEvent( new CustomEvent('loading-layer-hide', { detail: { id: 'global' } }) );

Multi-Phase Messages (by delay)

Messages change automatically at specified intervals from show().

Waiting for phases...
<loading-layer> <loading-phase message="Cargando tus datos..."></loading-phase> <loading-phase message="Parece que está tardando..." delay="3"></loading-phase> <loading-phase message="Esto tiene mala pinta..." delay="7" auto-close="5"></loading-phase> </loading-layer>

Multi-Phase Messages (by event)

Simulates 3 async tasks. Each dispatches a custom event when complete, advancing the phase.

Waiting for tasks...
<loading-layer id="loader"> <loading-phase message="Connecting..."></loading-phase> <loading-phase message="Fetching data..." event="task-connected"></loading-phase> <loading-phase message="Almost there..." event="task-data-loaded" auto-close="2"></loading-phase> </loading-layer> <script> loader.show(); await connectToServer(); document.dispatchEvent(new CustomEvent('task-connected')); await fetchUserData(); document.dispatchEvent(new CustomEvent('task-data-loaded')); // → auto-closes after 2 seconds </script>

Completion Event

loading-layer-complete fires when the layer hides, with reason and duration.

Click the button to see the completion event...
loader.addEventListener('loading-layer-complete', (e) => { console.log(e.detail.reason); // 'timeout' | 'manual' | 'event' | 'phases-complete' console.log(e.detail.duration); // ms elapsed since show() });
